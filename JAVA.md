What is Java? 
Java is a general-purpose, high-level, object-oriented, class-based programming language designed to have minimum implementation dependencies.
History:
Developed by James Gosling at Sun Microsystems in 1995
Initially called Oak
Later renamed as Java
Now maintained by Oracle Corporation
Nature of Java:
High-level → Easy for humans to read and write
Compiled + Interpreted
Compiled into bytecode
Interpreted by JVM
Object-oriented → Everything revolves around classes and objects
