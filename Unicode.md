Unicode Concept
What is Unicode?
Unicode is a universal character encoding system that supports all languages in the world.
Features:
Uses 16 bits
Can represent 65,536 characters
Supports:
English
Indian languages
Symbols
Emojis (extended)
Why Java Uses Unicode?
Java is a global language
Supports internationalization
Works on all platforms
Overcomes ASCII limitations
Unicode Example:
char ch = '\u0905'; // Hindi character
