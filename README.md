# Java Multithreading Examples

This repository contains basic examples of multithreading in Java.

## Programs Included

### 1. ThreadExample.java
- Uses Thread class
- Demonstrates parallel execution

### 2. RunnableExample.java
- Uses Runnable interface
- Better approach for multithreading

## How to Run

1. Compile:
   javac ThreadExample.java
   javac RunnableExample.java

2. Run:
   java ThreadExample
   java RunnableExample

## Output
- Two threads run simultaneously
- Output may vary due to thread scheduling
